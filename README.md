# AI-Powered Smart Learning Companion for Personalized Education

This project aims to develop an AI-powered smart learning companion designed to offer personalized educational experiences. The companion will dynamically adapt to individual user needs by providing tailored explanations, generating customized quizzes, and suggesting diverse learning strategies. It leverages advanced Large Language Models (LLMs) and various prompt engineering techniques to achieve a high degree of personalization.

The system is built as a standalone application running entirely on localhost, eliminating the need for external databases or persistent storage. Content generation, difficulty adaptation, and learning path recommendations are performed dynamically in real-time based on user interactions and predefined prompt types. This approach emphasizes an iterative development process from scratch, focusing on core AI functionalities and their integration into a responsive web interface.

The primary goal is to demonstrate the practical application of generative AI in EdTech, specifically in creating an adaptive learning environment. It explores the nuances of prompt engineering for different educational outcomes, integrating retrieval-augmented generation (RAG) for contextual accuracy, and presenting a user-friendly interface for an engaging learning experience.
